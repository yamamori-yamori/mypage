# 小説推敲 Diffビューア（novdiff）v1.0

小説の推敲過程で、**古い版**と**新しい版**を左右に並べて差分を視覚的に確認・編集できる、完全ローカル動作の単一HTMLツールです。
- 作者: [やまもりやもり](https://yamamori-yamori.github.io/mypage/)

- 外部サーバー・CDN不要。完全ローカルHTML。
- ブラウザでHTMLファイルを開くだけで動作します。Windows / macなどプラットフォームを問いません。
- 日本語長編小説を前提に設計されています。10万文字超えていても快適に動作します。

---

## 使い方

1. `novdiff.html` をブラウザで開く（Chrome / Edge / Firefox 推奨）
2. 左側に古いテキストの `.txt` をドラッグ＆ドロップ
3. 右側に新しいテキストの `.txt` をドラッグ＆ドロップ
4. 両方揃うと自動で差分が色付き表示される
5. 右側で推敲（編集）する
6. 右ペインからフォーカスを外すと、自動で差分が再計算・再表示される
7. 「保存」ボタンで新しいテキストを保存（OSの保存ダイアログ）

画面左端のサムネでスクロールできます。  
ツールバー右端の **by やまもりやもり** をクリックすると、作者ページが新しいタブで開きます。
