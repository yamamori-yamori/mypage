マンガページエディタ 使い方 by やまもりやもり

画面 手順 ステップ 選択 ページ コピー 出力 キー FAQ
Ver.4.4

● マンガのコマをレイアウトして画像を入れ、吹き出しとセリフを置けます。ネーム用下書きや校閲メモもあります。

● 複数ページにも対応。線のガサつきや手書き風の変形、多彩な吹き出し、集中線・スピード線も使えます。

1 画面の見方

上バーでファイルとページ、左で作業ステップ、中央で描画、右で設定とプロパティ。

図1 — 画面4領域。青い枠のステップが「今やっている作業」。下書きは暗いグレー、メモは赤。

左ステップ

作業に合わせてツールが切り替わります。紙を決める → コマを置く → 絵を入れる → 吹き出し → セリフ → エフェクト。
下書きでネーム、メモで赤入れもできます。

中央キャンバス

  • 原稿そのもの。空きをドラッグで作成、Shift で選択・移動
  • Ctrl/Cmd + ドラッグ … パン／Ctrl/Cmd + ホイール … ズーム
  • Ctrl/Cmd + C / V … コピー／貼り付け

右上ガイド

今のステップの説明と、台紙・下書きの種類などの設定。

右下プロパティ

選んでいるコマ／絵／吹き出しなどの数値と色。

2 おすすめの作業順

迷ったらこの流れ。下書きはいつでも挟めます。

図2 — 基本フロー。緑は任意の下書き。校閲のメモは最後に足す想定（図には未掲載）。

レイヤーのイメージ: 台紙色・台紙画像 → コマの塗り → 下書き → 絵 → 枠 → 吹き出し → セリフ → メモ（校閲）。 絵を載せると下書きは自然に隠れます。メモは常に一番上に見えますが、PNG・印刷・サムネには出ません。

3 各ステップでできること

左ナビの8段階。それぞれ「描く／置く」ツールが変わります。最後のメモは校閲専用です。

紙を決める

  • A5 / B5 / A4 / SNS / PC / スマホ縦など
  • 向き・カスタムサイズ
  • この画面に画像をドロップするとそのサイズになります

コマを置く

  • 空きでドラッグ → 新規コマ。頂点を掴んで変形
  • 斜め線も使えるグリッド吸着が強力でオススメ！
  • コマの融合や、線のガサつきも設定できる
  • 台紙色・台紙画像も選べる

絵を入れる

  • コマを選び画像ファイル選択
  • 画像のドラッグアンドドロップも可能
  • 移動・回転・拡大縮小・色調整

吹き出し

  • 形・しっぽはプロパティで変更
  • 複数選んで融合可
  • コマでクロップもできる（既定OFF）

セリフ

  • クリックして入力
  • 縦書き・横書き
  • アウトライン化して手書き風の変形も可能

エフェクト

  • 集中線・スピード線など
  • コマ内に配置
  • 選択後に移動・回転

下書きレイヤー（✎）

図3 — 下書き。印刷には出ない。PNG には含まれる。

セリフ / 下書きの文字列 / メモ の違い: 本文は「セリフ」ステップ。構図・作業用の灰色テキストは下書きの「文字列」。校閲の赤+白縁は左ナビ最下段の「メモ」です。

メモレイヤー（校閲・赤）

図3b — 校閲メモ。PNG・印刷・サムネには出ない。Ctrl/Cmd+A の全選択にも入らない。

操作

  • クリック … 文字メモ
  • ドラッグ … 赤い自由曲線
  • Shift … 選択・移動（ハンドルでの拡大縮小・回転なし）
  • 融合・コマクロップなし

一括削除

  • 右上ガイド:「このページのメモをすべて削除」
  • サムネ側:「全ページのメモを削除」
  • Undo で戻せます

4 選択・移動・融合

専用の「選択ステップ」はありません。Shift で一時的に選択モードになります。

図4 — 選択モードと融合。

操作結果
クリック選択（融合なら仲間ごと）
Shift+クリック追加／解除
空ドラッグ範囲選択
ドラッグ移動
Delete / Backspace削除（入力中は文字消し）
コマ内にクロップ吹き出し・下書きをコマ形で切る（既定OFF）。メモには無し
Ctrl/Cmd+A現在ページの通常オブジェクトを全選択（メモは対象外）

5 複数ページ

追加・削除・並べ替えはサムネ一覧だけ。上バーは前後移動とサムネを開く用。

図5 — ページ操作。最後の1枚は削除不可。最大64ページ程度。

Undo に注意: ページ追加の直後に「元に戻す」と、追加そのものが取り消されてページが消えることがあります。

6 コピー＆ペースト

画像データ込みでコピーできます。同じページと別ページで「ずれ方」が違います。

図6 — 同一ページは少しずらす／別ページは座標維持。

7 保存・PNG・印刷

作業の本体は .manga.json 保存（中身は JSON。識別タグ付き・無くても読込可）。PNG と印刷の対象範囲が違います。

図7 — 出力3種の違い。

表示の拡大

  • 上バーの倍率セレクト
  • Ctrl/Cmd + ホイール … カーソル基準ズーム
  • Ctrl/Cmd + ドラッグ … パン
  • ズームは見た目だけ（保存に入らない）

Undo

  • Ctrl/Cmd + Z / 上バー「元に戻す」
  • やり直しは Ctrl/Cmd + Y または Shift + Z
  • 履歴はおおよそ直近10操作
  • ページをまたいでも履歴は共有

8 キーボード早見

Windows は Ctrl、Mac は ⌘（Cmd）。文字入力・IME 変換中は一部キーが文字編集優先になります。

キー機能
Shift（押し続け）選択・範囲選択・ハンドル操作
Ctrl/Cmd + Z元に戻す
Ctrl/Cmd + Y / Shift+Zやり直し
Ctrl/Cmd + A現在ページの全選択（メモは除外）
Ctrl/Cmd + C / Vコピー / 貼り付け
Delete / Backspace選択削除（入力中は文字消し）
← ↑ → ↓選択中: 1px 移動。未選択: 画面スクロール
[ / ]前ページ / 次ページ
Ctrl/Cmd + ホイールズーム（見た目のみ）
Ctrl/Cmd + ドラッグパン（キャンバス移動）

9 よくある質問

「壊れた？」と思いやすい挙動の多くは仕様です。

下書きが絵の上に出ない

仕様です。下書きは画像の下。絵を載せると隠れます。

下書きステップでコマが選べない

仕様です。下書き作業中は下書きだけ触れます。

吹き出しがコマで切られる

「コマ内にクロップ」が ON になっていないか確認。既定は OFF。

印刷に下書きが残る？

印刷は下書きオフ。残るのは PNG 側です。校閲メモは PNG にも印刷にも出ません。

メモが PNG に出ない / Ctrl+A で選ばれない

仕様です。メモは編集画面だけの校閲レイヤーです。全選択からも外れます。

台紙がコピーできない

台紙は背景扱い。ページ追加の「同じ台紙」を使います。

融合したのに片方だけ動く

再選択して全員がハイライトされるか確認。だめなら解除→再融合。

1ページだけ書き出し・読み込みしたい

ブラウザを2枚開き、元で Ctrl/Cmd+A → C、先で V。コマレイアウトや棒人形はストック用 HTML にまとめてコピペも可。
